# Smart Junction Simulation

An Adaptive Traffic Control System developed using C++ and Object-Oriented Programming concepts.

## Overview

Traditional traffic signal systems often rely on fixed timing, which can cause unnecessary congestion and delays. This project simulates a smart traffic junction that dynamically manages traffic flow based on traffic density while also supporting emergency vehicle prioritization and automated violation detection.

## Features

* Density-based traffic signal scheduling
* Emergency vehicle prioritization
* Overspeed violation detection
* Automated fine generation
* Traffic flow simulation
* File-based data persistence
